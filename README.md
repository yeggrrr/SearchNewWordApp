# 🔎 신조어 검색 앱 만들기 🔎
##
## 신조어를 검색하는 앱입니다. 궁금한 신조어를 검색하면, 하단에 뜻이 출력됩니다. 또한 새로운 신조어를 추가할 수 있습니다.
- 검색기능 (재검색 시, textField가 초기회되로록 구현)
- 검색어 추천 버튼 (검색어를 클릭하거나, 검색 버튼을 클릭하면 랜덤으로 추천되도록 구현)
- 상단 + 버튼을 통해 추가하기 화면으로 이동
- 새로운 신조어 추가 기능 (추가완료 alert & 입력하지 않으면 추가되지 않도록 구현)
- keyboard return키 클릭 시, 키보드가 내려가도록 구현
